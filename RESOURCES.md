# Resources

## Primary (high-trust)
- **[React official docs — Learn](https://react.dev/learn)** — The canonical,
  maintained-by-the-React-team tutorial. Teaches modern function components +
  hooks. This is the default primary source for lessons.
  - [Your First Component](https://react.dev/learn/your-first-component)
  - [Writing Markup with JSX](https://react.dev/learn/writing-markup-with-jsx)
  - [Describing the UI (chapter)](https://react.dev/learn/describing-the-ui)

## JavaScript shore-up (referenced as gaps appear)
- **[MDN — JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)**
  — Trusted reference for JS language features (arrow functions, `.map`,
  destructuring, etc.).

## Communities (for wisdom / testing skills in the real world)
- **[r/reactjs](https://www.reddit.com/r/reactjs/)** — Active, high-signal.
- **[Reactiflux Discord](https://www.reactiflux.com/)** — Large real-time
  React community; good for quick questions and job-hunt advice.
