Components

A component is a JavaScript function that returns markup (JSX). Its name must start with a capital letter — that's how React tells your components apart from plain HTML tags.

function MyButton() {
  return <button>Click me</button>;
}

Nesting & exporting

Components render other components. export default marks the main component of a file.

export default function App() {
  return (
    <div>
      <h1>Welcome</h1>
      <MyButton />
    </div>
  );
}

JSX rules

  • Return one root element. Wrap siblings in a <div> or an empty Fragment <>...</>.
  • Every tag must close: <br />, <img />.
  • Self-closing components: <MyButton />.
